## Setup

Add .env file.
```env
HOST_NAME=*** # Connection info Database Host 
PASSWORD=***　# Connection info Password
PORT=*** # Connection info Port
```

## Run it
```
$ vr server
```
Then, open browser to `http://localhost:8000/`.
